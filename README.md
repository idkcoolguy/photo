# photo

Simple single‑page image editor demo.

## GitHub Pages

This repository can be published using GitHub Pages. To make the site available at:

```
https://idkcoolguy.github.io/photo/
```

just serve the `index.html` file from the repository root (which is the default when "Source" is set to `main` branch).

You can also enable Pages in the repo settings and select **main** / **root** as the source. After a minute or two the page will be live.

> You don’t need any build step; the HTML, CSS and JS are all self‑contained.
